# Ion‑Drive Clipper

## Overview
The Ion‑Drive Clipper is a small, low‑cost ship used for short trips within a single star system.

## Specifications
- Top Speed: 0.35c (fast sublight travel)
- Fuel: Ionized Xenon
- Seats: 12 passengers
- Artificial Gravity: No
- Faster‑Than‑Light Travel: None

## Features
- Basic cabin layout
- Standard safety shielding

## Notes
The Clipper focuses on efficiency, not comfort. It’s great for quick hops between local destinations.
